# University Admission FAQ Chatbot

A chatbot developed using Python, Streamlit, NLP, TF-IDF Vectorization, and Cosine Similarity.

## Features
- Admission FAQ answering
- Streamlit user interface
- Recommended questions
- Chat history sidebar
- TF-IDF and Cosine Similarity matching

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
